## BloodStory
고전 아케이드 게임인 팩맨에 액션 및 성장 요소를 추가해 ARPG라는 새로운 시각으로 접근한 게임입니다.

팩맨이 쿠키를 먹는 것과 파워 쿠키를 먹고 유령을 잡는 것에서 적혈구와 산소, 백혈구와 항원 관계를 연관시켜 의학적인 요소를 추가했습니다.
